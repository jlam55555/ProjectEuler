~Finds the sum of the diagonals of a 1001x1001 number spiral.~ #My algorithm:
5x5 spiral diagonals = 
1 + 
9 + 7 + 5 + 3 + 
25 + 21 + 17 + 13

which can also be represented as
1 +
3^2-0*(3-1) + 3^2-1*(3-1) + 3^2-2*(3-1) + 3^2-3*(3-1) +
5^2-0*(5-1) + 5^2-1*(5-1) + 5^2-2*(5-1) + 5^2-3*(5-1)

or, if simplified, you start with 1 and for each layer n you can add:
4*n^2-6*(n-1)

and this pattern can easily be continued.
